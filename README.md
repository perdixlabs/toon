# TOON Visualizer

Token-efficient JSON visualization and comparison tool.

## Features

- Compare JSON vs TOON token counts
- Support for GPT-4o, GPT-4, and GPT-3 tokenizers
- Real-time token breakdown visualization
- JSON repair and validation

## Development

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000)
